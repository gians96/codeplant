[Uso de NVM]

##Instalar node version 8 o la versión que quieras.
En la línea de comando, escribe el comando que hay más abajo. Si quieres verificar cuáles son las versiones de node actuales, puedes ir a nodejs.org y verlas todas. Nosotros te recomendamos usar la recomendada para la mayoría de los usuarios.

nvm install 8.15.0

##Verificar las versiones de node instaladas.
Siempre verifica las versiones de node que hay instaladas. Algunas veces nuestras aplicaciones no se ejecutan porque usamos versiones desactualizadas. Este comando te mostrará todas las versiones de node que tengas instaladas en Windows.

nvm list

Cambiar entre diferentes versiones de node.
Siempre puedes usar diferentes versiones de node y este comando te permite saltar entre todas tus versiones instaladas.

nvm use 8.15.0

o tambien:

nvm use 10.15.1
